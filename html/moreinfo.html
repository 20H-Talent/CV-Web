We train you in highly demanded skills.

A Front-End Developer is autonomous to perform layout work and is able to design and build the visible face of web applications (or Single-page applications).

You will be familiar with HTML5 and CSS3, responsive design, CSS pre-processors (SASS), CSS Frameworks (Bootstrap), JavaScript (ES6 +, DOM, events and AJAX), asynchronous programming (callbacks, promises and async / await), query RESTful API's, component-based design (React + Redux), task automation (gulp), team tools (git, GitHub, Slack and Trello), agile methodologies (SCRUM), etc ...

You will also strengthen fundamental professional skills, such as teamwork, self-learning or public exposure, among others.

We guide you during the incorporation into the company.

During the last stage of the training, you will know the challenges that the companies are facing and you will have the possibility to contribute your bit in a real project.
